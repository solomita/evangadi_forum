✉️ Email Confirmation API

Test the Evangadi Forum email confirmation endpoint

Endpoint: POST /api/auth/confirm-email
Base URL: http://localhost:5004
Purpose: Activate user account via confirmation token
1. Confirmation Token
📋 How it works:
  • User registers with email
  • Backend creates unverified user (is_verified = 0)
  • Confirmation token sent in email link or displayed on screen
  • User clicks link → token sent to this endpoint
  • Backend validates token and marks user as verified (is_verified = 1)
  • User can now sign in
